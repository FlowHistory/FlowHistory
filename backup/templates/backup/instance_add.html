Add Instance

Instances are configured via environment variables in your .env file. Add the variables below and restart the container. Env values seed settings on first discovery only.
{% include "backup/components/_card_section_start.html" with title="Local Instance (file-based)" extra_class="mb-3" %}

For Node-RED running on the same Docker host:

# Add to .env — replace PREFIX with a unique name (e.g., LOCAL, DEV)
FLOWHISTORY_PREFIX_FLOWS_PATH=/nodered-data/flows.json
FLOWHISTORY_PREFIX_NAME=My Node-RED
{% include "backup/components/_card_section_end.html" %} {% include "backup/components/_card_section_start.html" with title="Remote Instance (API-based)" extra_class="mb-3" %}

For Node-RED on another server:

# Add to .env — replace PREFIX with a unique name (e.g., PROD, STAGING)
FLOWHISTORY_PREFIX_URL=http://192.168.1.50:1880
FLOWHISTORY_PREFIX_USER=admin
FLOWHISTORY_PREFIX_PASS=yourpassword
FLOWHISTORY_PREFIX_NAME=Production
{% include "backup/components/_card_section_end.html" %} {% include "backup/components/_card_section_start.html" with title="Optional Settings" extra_class="mb-3" %} 
# Schedule
FLOWHISTORY_PREFIX_SCHEDULE=daily        # hourly, daily, weekly
FLOWHISTORY_PREFIX_TIME=03:00
FLOWHISTORY_PREFIX_DAY=0                 # day of week for weekly (0=Mon … 6=Sun)
FLOWHISTORY_PREFIX_ALWAYS_BACKUP=false

# Retention
FLOWHISTORY_PREFIX_MAX_BACKUPS=20
FLOWHISTORY_PREFIX_MAX_AGE_DAYS=30

# Appearance
FLOWHISTORY_PREFIX_COLOR=#3B82F6         # hex color for UI accent

# Remote polling
FLOWHISTORY_PREFIX_POLL_INTERVAL=60      # seconds

# Local-only options
FLOWHISTORY_PREFIX_WATCH=true            # watch for file changes (inotify)
FLOWHISTORY_PREFIX_DEBOUNCE=5            # seconds to wait before backup after change
FLOWHISTORY_PREFIX_BACKUP_CREDENTIALS=false  # include flows_cred.json
FLOWHISTORY_PREFIX_BACKUP_SETTINGS=false     # include settings.js
FLOWHISTORY_PREFIX_RESTART_ON_RESTORE=false  # restart container after restore
FLOWHISTORY_PREFIX_CONTAINER_NAME=nodered    # Docker container name for restart
{% include "backup/components/_card_section_end.html" %} {% include "backup/components/_card_section_start.html" with title="Notifications" extra_class="mb-3" %}

Per-instance overrides. Global defaults use FLOWHISTORY_NOTIFY_ prefix (see README). 

# Enable/disable and event filter
FLOWHISTORY_PREFIX_NOTIFY=true
FLOWHISTORY_PREFIX_NOTIFY_EVENTS=all     # or: backup_failed,restore_success,restore_failed

# Discord
FLOWHISTORY_PREFIX_DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...

# Slack
FLOWHISTORY_PREFIX_SLACK_WEBHOOK_URL=https://hooks.slack.com/services/...

# Telegram (both required)
FLOWHISTORY_PREFIX_TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
FLOWHISTORY_PREFIX_TELEGRAM_CHAT_ID=123456789

# Pushbullet
FLOWHISTORY_PREFIX_PUSHBULLET_API_KEY=o.abc123...

# Home Assistant (both required)
FLOWHISTORY_PREFIX_HOMEASSISTANT_URL=http://homeassistant.local:8123
FLOWHISTORY_PREFIX_HOMEASSISTANT_TOKEN=eyJ...
